ng-repeat | filter:

Create an app, with an array of data in the service that showns up on the screen as a list. Where you can type to filter what's shown in the list.

Outcome

{{alive}}

index.html

<input ng-model="livingFilter"></input>
<p ng-repeat="alive in living| filter: livingFilter">
    {{alive}}
</p>

contorler.js

$scope.living = myService.living;

service.js

this.living = ['Monkey', 'Dog', 'Cat', 'Mouse'];

$http

Create an app hitting an api (swapi.co, birdapi.com, pokeapi, smurfs, marvel api, etc)

Outcome

index.html

<div id="log1"></div>
<div id="log2"></div>

contorler.js

$scope.getUsers = myService.getUsers;

service.js

var xhr = new XMLHttpRequest();
xhr.open("GET", "http://smurfs.devmounta.in/smurfs/", false);
xhr.send();

document.getElementById("log1").innerHTML=(xhr.status);
document.getElementById("log2").innerHTML=(xhr.statusText);
